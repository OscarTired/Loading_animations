# CSS Loading Animations

![HTML5](https://img.shields.io/badge/html5-%23E34F26.svg?style=for-the-badge&logo=html5&logoColor=white)
![CSS3](https://img.shields.io/badge/css3-%231572B6.svg?style=for-the-badge&logo=css3&logoColor=white)

A collection of modern, pure CSS loading animations for buttons. Designed to be lightweight, performant, and easy to integrate into any web project.

## Features

- **8 Unique Effects**: Pulse, Glow, Shimmer, Spinner, Bouncing Dots, Border Fill, Gradient Flow, and Shrink & Spin.
- **Pure CSS**: No external libraries or heavy JavaScript dependencies.
- **Responsive**: Adapts to different screen sizes.
- **Modern Design**: Glassmorphism aesthetic with gradients and subtle shadows.

## Usage

1. Open `loading_animations.html` in your browser to view the gallery.
2. Click on any button to trigger its loading state.
3. Use the "Interactive Playground" to test all animations simultaneously.

> [!TIP]
> You can easily customize the colors using the CSS variables defined in the `:root` section of the style block.

> [!NOTE]
> All animations use hardware-accelerated properties (transform, opacity) for smooth performance.
